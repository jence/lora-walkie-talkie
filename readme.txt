This is simple arduino code to test of transmission of ebyte-e32 lora module.
The sender sends hello every 1 sec to receiver.

HOW TO USE

1.copy the LoRa librery from lib to documents/arduino/libraries
2.upload sender and receiver code to two differernt mcu.
3.receiver has nokia 5110 display so install Adafruit_GFX librery v1.11.9 and Adafruit_PCD8544 livrery v2.0.3
